How to Build Level, Energy, and Health Systems for Your Game in SandMod Engine

1. Introduction :

In this tutorial, weâ€™ll create a simple Player Stats System that includes:
1. Health (when damaged or healed)
2. Energy (used when running or performing actions)
3. Level & XP (progression system)

First we gonna start gonna start by making a typescript file to make our playerstats class :

after Creating the typescript file rename it to PlayerStats and then double click to open the file in vsc (recommended) if u don't have vsc installed in your system please download it from here 

Now we can edit the typescript file to make the health, energy, level systems we will start by making the health system :

First we gonna make a class member to store the health data first we make these two members :

    public Health: number = 0;
    private MaxHealth: number = 100;

These two are going to store the player current health and the player max health After this we create two functions one for taking damage and one for healing But before that we need to understand how the healing effect works It can be instant or per second

For example when you drink a potion in a game that heals 5 HP you want the player to instantly gain that health So in this case you use the first parameter of the Heal function to instantly recover HP

But if you want the player to recover HP slowly over time like a regeneration effect then you use the other parameters which allow the player to recover HP per second for a specific duration

This means the Heal function can work in two ways
1: Instant heal adds HP immediately
2: Gradual heal recovers HP every second

this is the function to actually make that first we make this members then we make the damage and heal functions:
    private healingTimer: number = 0;
    private healingActive: boolean = false;
    private healAmountPerSecond: number = 0;
    Damage(amount : number): void {
        if (this.Health > 0) {
            this.Health -= amount;
        }
    }

    Heal(amount: number, recoverpersecond: boolean = false,
         recoverpersecondamount: number = 0.5,
          recoverpersecondduration: number = 2): void {
        if (this.Health <= this.MaxHealth) {
            if (!recoverpersecond) {
                this.Health += amount;
            } else {
                this.healingActive = true;
                this.healingTimer = recoverpersecondduration;
                this.healAmountPerSecond = recoverpersecondamount;
            }
        }
    }

now the damage and heal functions are ready one last step to make the recovering per second works is to edit the OnUpdate function :

    OnUpdate(): void {
        if (this.healingActive) {
            if (this.healingTimer > 0) {
                this.Health += this.healAmountPerSecond * Time.deltaTime;
                if (this.Health > this.MaxHealth) this.Health = this.MaxHealth;

                this.healingTimer -= Time.deltaTime;
            } else {
                this.healingActive = false;
            }
        }
    } 

now we have made the Health system next is the Energy System :

Now lets make the Energy System
The Energy System works similar to the Health System but instead of controlling the player life it controls the player stamina or power used for actions like sprinting jumping or attacking

We will start by creating two variables just like we did for health
this.MaxEnergy and this.Energy
These two will store the player maximum energy and current energy value
    public Energy: number = 0;
    private MaxEnergy: number = 100;

Now we create two main functions one for using energy and one for recovering it

The UseEnergy(amount: number) function is used to reduce the player energy when performing actions for example when the player sprints or jumps you can call this function and pass how much energy it should consume

The RecoverEnergy(amount: number, recoverPerSecond: boolean = false, recoverPerSecondAmount: number = 0.5, recoverPerSecondDuration: number = 2) function works the same way as the Heal function from the Health System It can either instantly recover energy or slowly recover it per second

    private EnergyRecoverTimer: number = 0;
    private EnergyRecoverActive: boolean = false;
    private EnergyRecoverAmountPerSecond: number = 0;
    UseEnergy(amount: number): void {
        if (this.MaxEnergy > 0) {
            this.Energy -= amount;
        }
    }

    RecoverEnergy(amount: number, recoverPerSecond: boolean = false,
         recoverPerSecondAmount: number = 0.5,
          recoverPerSecondDuration: number = 2): void {
        if (this.Energy <= this.MaxEnergy) {
            if (!recoverPerSecond) {
                this.Energy += amount;
            } else {
                this.EnergyRecoverActive = true;
                this.EnergyRecoverTimer = recoverPerSecondDuration;
                this.EnergyRecoverAmountPerSecond = recoverPerSecondAmount;
            }
        }
    }

now we edit the OnUpdate function the same way we did to Healing effect :

        if (this.EnergyRecoverActive) {
            if (this.EnergyRecoverTimer > 0) {
                this.Energy += this.EnergyRecoverAmountPerSecond * Time.deltaTime;
                if (this.Energy > this.MaxEnergy) this.Energy = this.MaxEnergy;

                this.EnergyRecoverTimer -= Time.deltaTime;
            } else {
                this.EnergyRecoverActive = false;
            }
        }

We will start by creating three variables
this.Level this.XP and this.RequiredXP
The Level variable stores the player current level
XP stores how much experience the player currently has
RequiredXP stores how much XP is needed to level up

Now we create a function called AddXP(amount: number)
This function will increase the player XP when they earn experience
When the XP becomes equal or higher than RequiredXP the player levels up
    public Level: number = 0;
    public XP: number = 0;
    private RequiredXP: number;

and then we create the functions :

    AddXP(amount: number): void {
       this.XP += amount;
       if (this.XP >= this.RequiredXP) {
          this.LevelUp();
       }
    }

    LevelUp(): void {
       this.Level += 1;
       this.XP -= this.RequiredXP;
       this.RequiredXP = Math.floor(this.RequiredXP * 1.5);
    }

What this does is:

1. Increase the player level by 1

1. Remove the XP that was used to level up

1. Increase the XP needed for the next level by 50 percent

Final :

Final Code :

class PlayerStats
    extends Component {

    public Health: number = 0;
    private MaxHealth: number = 100;
    public Energy: number = 0;
    private MaxEnergy: number = 100;
    public Level: number = 0;
    public XP: number = 0;
    private RequiredXP: number;

    private healingTimer: number = 0;
    private healingActive: boolean = false;
    private healAmountPerSecond: number = 0;

    private EnergyRecoverTimer: number = 0;
    private EnergyRecoverActive: boolean = false;
    private EnergyRecoverAmountPerSecond: number = 0;

    OnStart(): void {
        Debug.Log("OnStart");
    }

    OnUpdate(): void {
        if (this.healingActive) {
            if (this.healingTimer > 0) {
                this.Health += this.healAmountPerSecond * Time.deltaTime;
                if (this.Health > this.MaxHealth) this.Health = this.MaxHealth;

                this.healingTimer -= Time.deltaTime;
            } else {
                this.healingActive = false;
            }
        }

        if (this.EnergyRecoverActive) {
            if (this.EnergyRecoverTimer > 0) {
                this.Energy += this.EnergyRecoverAmountPerSecond * Time.deltaTime;
                if (this.Energy > this.MaxEnergy) this.Energy = this.MaxEnergy;

                this.EnergyRecoverTimer -= Time.deltaTime;
            } else {
                this.EnergyRecoverActive = false;
            }
        }
    }

    AddXP(amount: number): void {
       this.XP += amount;
       if (this.XP >= this.RequiredXP) {
          this.LevelUp();
       }
    }

    LevelUp(): void {
       this.Level += 1;
       this.XP -= this.RequiredXP;
       this.RequiredXP = Math.floor(this.RequiredXP * 1.5);
    }

    UseEnergy(amount: number): void {
        if (this.MaxEnergy > 0) {
            this.Energy -= amount;
        }
    }

    RecoverEnergy(amount: number, recoverPerSecond: boolean = false,
         recoverPerSecondAmount: number = 0.5,
          recoverPerSecondDuration: number = 2): void {
        if (this.Energy <= this.MaxEnergy) {
            if (!recoverPerSecond) {
                this.Energy += amount;
            } else {
                this.EnergyRecoverActive = true;
                this.EnergyRecoverTimer = recoverPerSecondDuration;
                this.EnergyRecoverAmountPerSecond = recoverPerSecondAmount;
            }
        }
    }

    Damage(amount : number): void {
        if (this.Health > 0) {
            this.Health -= amount;
        }
    }

    Heal(amount: number, recoverpersecond: boolean = false,
         recoverpersecondamount: number = 0.5,
          recoverpersecondduration: number = 2): void {
        if (this.Health <= this.MaxHealth) {
            if (!recoverpersecond) {
                this.Health += amount;
            } else {
                this.healingActive = true;
                this.healingTimer = recoverpersecondduration;
                this.healAmountPerSecond = recoverpersecondamount;
            }
        }
    }
}

You can use the code as follows :

First add this script as a Component to your player GameObject in Sandmod Engine
This Component controls everything related to the player stats like Health Energy and XP

After attaching the script you can call the functions inside it from any other script to control your player stats

now here is how you can use the functions :

    OnStart(): void {
        this.Damage(10); // damage 10 hp
        this.Heal(10); // instant heal 10 hp
        this.Heal(0, true, 1, 5); // recover 1 hp every 1sec for 5 seconds
        this.UseEnergy(10); // user 10 energy
        this.RecoverEnergy(10); // instant recover 10 energy
        this.RecoverEnergy(0, true, 5, 10); // recover 5 energy every 1sec for 10 seconds
        this.AddXP(100); // Add 100 xp
        this.LevelUp(); // level up
        this.RequiredXP = 100; // example xp needed for level 1
    } 

Now you can use the player stats class in your projects have fun!
