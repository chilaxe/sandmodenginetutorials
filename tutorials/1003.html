How to Create Voxel Maps for Sandmod Engine

Step 1 : Create Your Map in Minecraft

First, open Minecraft Java Edition and create a new world.
-- For maps such as BedWars, EggWars, or other custom arenas, it is recommended to use a Superflat world type so you have a clean building space.

After creating the world, build your map in Minecraft exactly how you want it to appear in your game.

Step 2 : Download the Export Tool

Once your map is finished, go to the Mineways website and download the program.

After downloading:

1 - Extract (unzip) the file.
2 - Launch the .exe file to open the software.

You will use this tool to export your Minecraft world as a 3D model.

After launching the application, go to the top menu and click:
File --> Open World
You will see a list of available Minecraft worlds. Select the world you created earlier and open it.

Once the world loads, you will see the map in the viewer.
  -- Move around: Hold the left mouse button and drag to navigate.
  -- Select an area: Hold the right mouse button and drag to select the region you want to export.
Make sure you select the entire map before continuing to the export step.

Once you have completed all the previous steps, click File again and select Export for Rendering.

After completing these steps and choosing a name for your file, a settings window will appear like the one shown below. Apply the same settings that I used then hit ok.

Step 3 : Import the file into blender 

Now for the final steps, import the file we just created into Blender :

If you’re not sure how to import the .obj file, you can do it in two simple ways in Blender:
1 - Drag and drop: Open Blender and drag the .obj file directly from your file explorer into the viewport.
2 - Import menu: Go to File --> Import --> Wavefront (.obj), then select your file.

After this you can simply click file --> export --> then .fbx and this menu will pop up follow the settings as it is here :

Final : Import to Sandmod Engine

After exporting the .fbx file, you can now import it into Sandmod Engine just like I showed in the previous tutorials.

And that’s it your map has been successfully imported into Sandmod Engine!
